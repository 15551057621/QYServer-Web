## 第三方资源

本项目使用了以下第三方库与服务：

### 图标库
- **Font Awesome 7.2.0** — 图标库  
  - 官网：https://fontawesome.com  
  - 许可证：Font Awesome Free License（包含 **SIL OFL 1.1** 与 **MIT License**）  

### 图表库
- **Chart.js** — 数据可视化图表库  
  - 官网：https://www.chartjs.org  
  - 许可证：MIT License

### Markdown 解析
- **marked.js** — Markdown 解析器  
  - 官网：https://marked.js.org  
  - 许可证：MIT License

### 服务器状态检查
- **minecraft-server-util** — Minecraft 服务器状态查询  
  - NPM：https://www.npmjs.com/package/minecraft-server-util  
  - 许可证：MIT License

### 服务器规则引用

  - 官网：https://twikoo.js.org/
